# Hello World

**"Hello, World!"** is a traditional first program in any programming language. It serves as a simple introduction to a language's syntax and environment.

## Examples

### Python
```python
print("Hello, World!")
```

### JavaScript
```javascript
console.log("Hello, World!");
```

### HTML
```html
<h1>Hello, World!</h1>
```

### Bash
```bash
echo "Hello, World!"
```

## History

The tradition dates back to 1972, when Brian Kernighan used "Hello, World!" in a tutorial for the B programming language. It was later popularized by Kernighan and Ritchie's book *The C Programming Language*.

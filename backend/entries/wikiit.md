# wikiit

**wikiit** is a warm-toned wiki encyclopedia built with [Django](/wiki/Django) REST Framework and [React](/wiki/HTML) + TypeScript, containerized with Docker and deployable on Kubernetes.

## Features

- **Create & Edit** — Write entries using Markdown with live preview and syntax highlighting
- **Search** — Full substring search across all entries
- **Random Page** — Explore the wiki by jumping to a random entry
- **Dark Mode** — Toggle between warm light and dark themes
- **Responsive** — Collapsible sidebar works on mobile and desktop

## Technology Stack

| Layer | Technology |
|-------|-----------|
| Backend | Django 6.0, Django REST Framework |
| Frontend | React 18, TypeScript, Vite |
| Markdown | CodeMirror 6, react-markdown |
| Container | Docker, docker-compose |
| Orchestration | Kubernetes |

## About

This project was built as a portfolio piece, demonstrating full-stack development with modern tooling. The warm color palette (terracotta, cream, earth tones) is inspired by natural materials — a deliberate departure from the cold blues common in developer tools.

# HTML

**HTML (HyperText Markup Language)** is the standard markup language for creating web pages. It describes the structure of a web page semantically using elements represented by tags.

## Basic Structure

```html
<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>This is a paragraph.</p>
</body>
</html>
```

## Common Elements

| Element | Purpose |
|---------|---------|
| `<h1>` to `<h6>` | Headings |
| `<p>` | Paragraphs |
| `<a>` | Links |
| `<img>` | Images |
| `<ul>` / `<ol>` | Lists |
| `<div>` | Division / Container |
| `<table>` | Tabular data |

## Semantic HTML

Modern HTML5 introduced semantic elements that clearly describe their meaning:

```html
<header>Site header</header>
<nav>Navigation links</nav>
<main>
  <article>Primary content</article>
  <aside>Related content</aside>
</main>
<footer>Site footer</footer>
```

## A Brief History

- **1991** — Tim Berners-Lee invents HTML
- **1995** — HTML 2.0 standardizes core features
- **1997** — HTML 4.0 introduces CSS integration
- **2014** — HTML5 becomes a W3C recommendation
